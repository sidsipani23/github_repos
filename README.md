# GitHub Repos

This is a simple project which provides an interface to search/query publically
hosted repositories of GitHub.

Tools/Packages used:

- React.js
- TypeScript
- Axios
- React Bootstrap
- react-infinite-scroll-component

## For a quick start:

- Clone the repository

```
   git clone https://github.com/sidsipani23/github_repos
```

- Install dependencies

```
   npm install
```

- Create the .env.development file with the keys listed as:

  1. VITE_GITHUB_KEY
  2. VITE_GITHUB_BASE_URL

- Run the project (In Development Mode)

```
   npm run dev
```

## To run the project in Production Mode:

- Create the equivalent .env.production file as above with Production
  credentials
- Build the project:

```
   npm run build
```

- Serve the production build locally using the command:

```
   npm run preview
```
